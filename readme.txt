Proyecto: Gestor de Noticias
Autor: Curso NAscor
Fecha: 2025
Descripción: CRUD de noticias con PHP, PDO y diseño responsivo básico.
Estructura:
- css/style.css : estilos principales
- img/ : carpeta de imágenes subidas
- partials/ : header y footer reutilizables
- index.php : listado principal
- anadir_noticia.php : formulario de creación
- editar_noticia.php : formulario de edición
- eliminar_noticia.php : elimina una noticia por ID
- noticia.php : vista detalle de una noticia
- conexion.php : conexión PDO a MySQL
